## Generation pipeline

```
python quality_judge.py   # Pillar 1 — G-Eval filter → dataset1240.clean.json
python eda.py             # Pillar 2 — EDA charts    → visuals/*.png
python split.py           # Pillar 3 — Stratified split → train/test_dataset.json
python report.py          # Pillar 4 — Summary docs  → slide3_summary.md
```

## Notes
- Hard-negative records (`example_type=hard_negative`) have empty `labels=[]` and are included in all charts.
- Hebrew tick labels rendered via `arabic_reshaper` + `python-bidi` for correct RTL display.
